Welcome to Map Matching Project pages.

A GPS track is a set of points with coordinates. This is usually generated by a location sensor as GPS sensor, mobile phone and so on. The points could be shifted from the road and a correction should be done.

The main goal of this project is to apply these corrections on a track. That includes :

  • a map matching process for a batch of GPS track
  • a map matching process for real time acquisition
  • a graphical user interface for 3d data visualization (tracks, networks, map)
